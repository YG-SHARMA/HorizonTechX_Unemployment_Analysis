# HorizonTechX_Unemployment_Analysis
# 📊 Unemployment Analysis Using Python

## 📌 Project Overview

This project analyzes unemployment trends in India using Python. The analysis focuses on understanding unemployment patterns across different states, identifying the impact of the COVID-19 pandemic, and generating insights through data visualization and exploratory data analysis (EDA).

The project demonstrates the use of Python libraries such as Pandas, NumPy, Matplotlib, and Seaborn for data cleaning, analysis, and visualization.

---

## 🎯 Objectives

* Clean and preprocess the unemployment dataset.
* Perform exploratory data analysis (EDA).
* Analyze unemployment trends across different states and regions.
* Study the impact of COVID-19 on unemployment rates.
* Identify correlations between unemployment, employment, and labour participation.
* Create informative visualizations.
* Generate insights that support economic and policy-level decision-making.

---

## 📂 Dataset

**Dataset:** Unemployment in India

The dataset contains information such as:

* Region (State/UT)
* Date
* Frequency
* Estimated Unemployment Rate (%)
* Estimated Employed
* Estimated Labour Participation Rate (%)
* Area (Urban/Rural)

## 📊 Key Insights

* Unemployment increased significantly during the COVID-19 pandemic.
* Urban areas generally experienced higher unemployment than rural areas.
* Certain states consistently recorded higher unemployment rates.
* Labour participation and unemployment rates showed noticeable variations across regions.
* Data visualization revealed seasonal and regional employment patterns.
